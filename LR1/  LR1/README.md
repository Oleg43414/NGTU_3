# Проект: Структуры данных на C++

Учебный проект, реализующий классические структуры данных с нуля.

## Структура проекта

```
include/include/     - Заголовочные файлы (.h)
src/src/            - Файлы реализации (.cpp)
Makefile            - Скрипт сборки проекта
```

## Реализованные структуры данных

1. **Массив** (Massiv) - Динамический массив с автоматическим изменением размера
2. **Односвязный список** (List_1) - Список с навигацией в одном направлении
3. **Двусвязный список** (List_2) - Список с двунаправленной навигацией
4. **Стек** (Steck) - LIFO структура на основе массива
5. **Очередь** (Queue) - FIFO структура на основе STL queue
6. **Full Binary Tree** (FullBinaryTree) - Полное бинарное дерево

## Компиляция и запуск

### Вариант 1: Использование Makefile (рекомендуется)

```bash
# Компиляция проекта
make

# Компиляция и запуск
make run

# Очистка скомпилированных файлов
make clean

# Справка по командам
make help
```

### Вариант 2: Ручная компиляция

#### Windows (PowerShell):
```powershell
g++ -o program.exe src/src/main.cpp src/src/Massiv.cpp src/src/List_1.cpp src/src/List_2.cpp src/src/Steck.cpp src/src/Queue.cpp src/src/FullBinaryTree.cpp -std=c++11 -Iinclude/include
.\program.exe
```

#### Linux/Mac:
```bash
g++ -o program src/src/main.cpp src/src/Massiv.cpp src/src/List_1.cpp src/src/List_2.cpp src/src/Steck.cpp src/src/Queue.cpp src/src/FullBinaryTree.cpp -std=c++11 -Iinclude/include
./program
```

## Работа с файлами

Программа автоматически сохраняет и загружает данные из файлов:

- `array_data.txt` - данные массива
- `singly_list_data.txt` - данные односвязного списка
- `doubly_list_data.txt` - данные двусвязного списка
- `stack_data.txt` - данные стека
- `queue_data.txt` - данные очереди
- `tree_data.txt` - данные дерева

При любых изменениях данные автоматически сохраняются в соответствующий файл.

## Возможности

### Массив
- Добавление элемента (в конец или по индексу)
- Получение элемента по индексу
- Удаление элемента по индексу
- Замена элемента по индексу
- Получение длины массива
- Вывод всего массива

### Односвязный список
- Добавление: в голову, в хвост, после элемента, перед элементом
- Удаление: из головы, из хвоста, по значению
- Поиск элемента по значению
- Вывод списка

### Двусвязный список
- Добавление: в голову, в хвост
- Удаление: из головы, из хвоста, по значению
- Поиск элемента по значению
- Вывод списка (вперёд и назад)

### Стек
- Push (добавление)
- Pop (удаление)
- Peek (просмотр верхнего элемента)
- Проверка на пустоту
- Получение размера
- Вывод стека

### Очередь
- Push (добавление)
- Pop (удаление)
- Front (просмотр первого элемента)
- Проверка на пустоту
- Получение размера
- Вывод очереди

### Full Binary Tree
- Добавление элемента (с сохранением структуры)
- Поиск элемента
- Проверка на Full Binary Tree
- Вывод дерева (In-order и Level-order обходы)

## Требования

- Компилятор C++ с поддержкой стандарта C++11 или выше (g++, clang++, MSVC)
- Make (опционально, для использования Makefile)
